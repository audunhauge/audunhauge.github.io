Datamaskinens oppbygging og virkemåte

Prosessor

Intel

Amerikansk produsent av prosessorer. Grunnlagt 1968 - laget intel 4004 CPU, 4 bits cpu. Dagens CPUer er 64bit. 4 bit gir 2^4 som største tall = 16. For 5-6 år siden var standard størrlese 32 bit (tall på 4294967296) Dagens CPU kan jobbe med tall på 2^64 = 18446744073709552000.

AMD

AMD (Advanced Micro Devices) laget sin første CPU 8080 AM9080 (en klone av intel 8080).

ARM

Acorn (Advanced) RISC (Reduced Instruction Set Computer) Machine. RISC CPUer har få og enkle instruksjoner (i motsetning til Intel). Hver instruksjon skal ta minst mulig tid. Kompliserte beregninger består av mange instruksjoner. Brukes i mobiler (nesten alle?)

Prosessoren er hjernen i datamaskinen. Hastigheten på en CPU er mellom 2 - 4 GHz for nye maskiner.

RAM og ROM

RAM er en forkortelse som betyr Random Access Memory

Lagring

Standard lagringsmedia i dag er SSD. Tidligere (for få år siden) var HDD det mest brukte.
